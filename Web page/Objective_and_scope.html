Objectives Of Projects


In this thesis, we use machine learning techniques on Bitcoin Price data to achieve the following objectives:

  1. Here we analyse, previous year data, to identify the important variable which affect the price. By using different curves visualization of data become easy.

  2. We use different machine learning algorithm and compare their accuracy rate, in order to achieve more accurate model.

  3. User can choose particular algorithm to predict the future price of Bitcoin.


Project scope


In this thesis, we will use the dataset obtained from python library, which provide as current modified data .We are using current data because of the intrinsic nature of the available data which contains a mix of technical and tactical performance indicators only. Since we know the outcome of the entire bitcoin price, this machine learning task will be supervised. We will do extensive pre-processing to modify the prepared current dataset for the suitability of each experiment. We will use many machine learning algorithms to get best and accurate model.
